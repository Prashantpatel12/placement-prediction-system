# AI Placement Prediction System

A professional AI-powered dashboard for predicting student placement probability using Machine Learning.

## 🚀 Features
- **Modern Dark-Mode UI**: Built with CustomTkinter.
- **Machine Learning**: Uses Random Forest Classifier for high-accuracy predictions.
- **Real-time Analytics**: Dashboard with charts and statistics.
- **Improvement Suggestions**: Personalized career advice based on student scores.
- **Data Export**: Export student records to CSV.
- **Database Integration**: SQLite for persistent storage.

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd "placement prediction"
   ```

2. **Install dependencies:**
   ```bash
   pip install customtkinter pandas numpy scikit-learn matplotlib pillow
   ```

3. **Run the application:**
   ```bash
   python main.py
   ```
